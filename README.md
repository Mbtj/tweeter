# Tweeter Project

Tweeter is a simple, single-page Twitter clone created to practice and demonstrate the use of the following tools:
- AJAX
- jQuery
- HTML5
- CSS3
- and Back-end code provided by Lighthouse Labs.

## Dependencies

- Express
- Node 5.10.x or above
